# robosys_device_driver
ロボットシステム学の課題で作ったものです。 LEDを点灯、消灯するデバイスドライバーを作成しました。

# 実行方法
    $ git clone 
    $ cd robosys_device_driver
    $ make
    $ sudo insmod myled.ko
    $ sudo chmod 666 /dev/myled0
    

# 映像
